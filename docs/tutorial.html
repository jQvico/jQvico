Tutorial

This is a quick tutorial, how to start work with jQvico.
For more complex solution go over to the Documentation section.

Install

Clone or download project from github. 

git clone https://github.com/jQvico/jQvico.git
jQvico is also available as npm and bower package. 

    npm install jqvico
    bower install jqvico

Router

Set up routing in app/config/router.js file.
For example for url http://localhost/myapp/#contact
Router can be the following:

    class Router{

        constructor(){
            this.classController = new Controllers();
        }

        getRouter(){
            return [
                //  contact form page
                {contact: [
                    {method: 'formController', class: this.classController},
                ]}
            ];
        }
    }

Controller

Create function formController() in app/controllers/controllers.js file. 

    class Controllers extends Controller{

        constructor(){
            super();
        }

        formController(){

            var data = {
                nameLabel: 'First name',
                textLabel: 'Message',
                sendLabel: 'Send message'
            };

            var callback = () => {
            };

            this.view.render('contact.html', $('main'), data, callback);
        }
    }

View

Create file contact.html which will be injected to section main in index.html template. 

    <form>
        <label>{{ data.nameLabel }}</label>
        <input name="firstName" type="text">
        <label>{{ data.textLabel }}</label>
        <textarea name="message"></textarea>
        <button type="submit">{{ data.sendLabel }}</button>
    </form>
